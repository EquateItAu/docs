title: "Setting tutor availability"
description: "Tutors set their weekly available hours so parents and students can find and book open slots."
---

## Overview

Availability is set by each **tutor** individually — admins cannot set it on their behalf. Once a tutor has published their availability, those slots appear in the booking calendar for parents and students.

## Steps (for tutors)

### 1. Open Calendar Settings

Log in as a tutor. From the sidebar, click **Settings**, then select the **Calendar** tab (or navigate to `/settings/calendar`).

### 2. Set your weekly availability

You'll see a weekly grid. Click and drag to mark the hours you're available each day.

Tips:
- Set the hours you're *generally* available, not the hours you're already booked — ClassQuill automatically hides slots that are already taken.
- You can set different hours for different days (e.g. available Mon/Wed/Fri afternoons, Sat mornings).
- Leave days unchecked if you never work that day.

### 3. Save

Click **Save availability**. Your slots are immediately visible to parents and students in the booking calendar.

## Blocking time off

If you need to be unavailable during a period that falls inside your usual availability (e.g. a holiday), you can block individual dates from your **Calendar** page — click any slot and select **Block time**.

<Warning>
  Blocking time does not cancel sessions already booked in that slot. Cancel those sessions separately if needed.
</Warning>

## What students and parents see

Once a tutor has set availability, parents booking through the parent portal and students on the **Find Tutor** page will see a calendar showing only open, unbooked slots for that tutor.
